# Автоматизация работы с маркетплейсами

Эта система помогает автоматически обновлять информацию о часах Casio на двух площадках: Ozon и Яндекс.Маркет.

## Что делает система

### Для Ozon
- Сама загружает актуальные данные о часах с сайта поставщика
- Обновляет количество доступных товаров
- Меняет цены согласно прайс-листу

### Для Яндекс.Маркета
- Работает с двумя способами продажи товаров:
  - **FBS** — когда товары хранятся на складе продавца и он сам отправляет заказы
  - **DBS** — когда продавец самостоятельно организует доставку
- Обновляет остатки отдельно для каждого способа продажи
- Синхронизирует цены с данными поставщика
- Управляет разными складами для FBS и DBS

## Откуда берутся данные

Система получает информацию напрямую с сайта официального поставщика Casio. Каждый раз она:
1. Скачивает свежий файл с остатками и ценами
2. Обрабатывает его
3. Отправляет изменения на маркетплейсы

**Важно:** Оба скрипта используют одни и те же данные от поставщика. Это гарантирует, что цены и остатки везде одинаковые.

## Польза для бизнеса
- Экономит время на ручном обновлении
- Исключает ошибки при переносе данных
- Цены и остатки всегда актуальные
- Поддерживает две модели продаж на Яндекс.Маркете
- Автоматически обнуляет остатки товаров, которых нет у поставщика

## Как работает с Яндекс.Маркетом

### Для FBS (товары на складе продавца)
1. Проверяет, какие товары размещены в кабинете FBS
2. Сравнивает с данными от поставщика
3. Обновляет остатки на складе FBS
4. Корректирует цены

### Для DBS (доставка силами продавца)
1. Проверяет товары в кабинете DBS
2. Синхронизирует остатки отдельно
3. Обновляет цены

### Что происходит с товарами
- Если товар есть у поставщика и на маркетплейсе — обновляется количество и цена
- Если товар есть на маркетплейсе, но закончился у поставщика — остаток становится "0"
- Если товар есть у поставщика, но не размещен на маркетплейсе — ничего не делается

## Безопасность и надежность
- Все изменения отправляются небольшими партиями
- Система обрабатывает ошибки соединения
- При проблемах с интернетом можно запустить повторно
- Данные никогда не теряются
