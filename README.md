# Автоматизация работы с маркетплейсами

Эта система помогает автоматически обновлять информацию о часах Casio на двух площадках: Ozon и Яндекс.Маркет.

## Что делает система

### Для Ozon
- Сама загружает актуальные данные о часах с сайта поставщика
- Обновляет количество доступных товаров
- Меняет цены согласно прайс-листу

### Для Яндекс.Маркета
- Работает с двумя способами продажи товаров (FBS и DBS)
- Обновляет остатки отдельно для каждого способа
- Синхронизирует цены с данными поставщика

## Откуда берутся данные

Система получает информацию напрямую с сайта официального поставщика Casio. Каждый раз она:
1. Скачивает свежий файл с остатками и ценами
2. Обрабатывает его
3. Отправляет изменения на маркетплейсы

## Польза для бизнеса
- Экономит время на ручном обновлении
- Исключает ошибки при переносе данных
- Цены и остатки всегда актуальные
- Работает с двумя моделями на Яндекс.Маркете

## Как работает
1. Проверяет, какие товары уже размещены на площадках
2. Сравнивает с данными от поставщика
3. Отправляет только необходимые изменения
4. Если товара нет у поставщика — показывает ноль в остатках
